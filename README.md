# ServerTemplate
Template server for other projects
# Clone: 
```
git clone --recurse-submodules https://github.com/Cadovvl/ServerTemplate.git
```
# Compile:

```
cd ServerTemplate
mkdir build
cd build
cmake ..
cmake --build .
```


# Run
```
./Server
```
